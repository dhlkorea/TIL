## 진행순서
1. Github 로컬 디렉토리에 코드작성 - 변경내역
2. 변경내역을 index에 올리기 -생성,수정,삭제
3. 로컬에 Commit하기 - summary / description 작성
4. Github Desktop에서 누르면 바로 Fetch누르면 Repository로 전송된다.
