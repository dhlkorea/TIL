## ※ 문제

terminal에서 빠르게 command를 지우고 싶다.



## ※ 해결

단축키를 활용한다

- `CTRL` + `C` : 현재 command를 취소한다.
- `CTRL` + `U` : 현재 command를 모두 지운다.
- `CTRL` + `A` : cursor를 문장 제일 앞으로 이동시킨다
- `CTRL` + `K` : 현재 cursor의 뒷부분을 모두 지운다.
