# 1.웹크롤링 시작하기 Kickoff
-----
## 크게 3가지 구현 방식 : 같은 모습의 웹페이지라도 구현이 모두 다르다!
* 단순 HTML 크롤링
  + 필요한 컨텐츠가 HTML을 통해 모두 표현되어 있을 경우
  + 가장 쉽고 빠른 처리
* Ajax 렌더링 크롤링
  + 필요한 컨텐츠가 서버로부터 JSON응답을 받아서 처리할 경우
* 자바스크립트 렌더링 크롤링
  + 필요한 컨텐츠가 JavaScript를 통해 처리할 경우
  + 제일 난감한 경우
